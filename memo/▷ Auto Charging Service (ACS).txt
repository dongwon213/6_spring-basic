▷ Auto EV Charging Service (AECS)
	
▶ 서비스 기획 의도
	- 자율주행 서비스를 이용하여 충전소로 이동하여 충전시간 단축
	- 전기차가 앞으로 많아짐에 따라 충전소가 많이 필요
	- 충전소를 지을 때 많은 비용 필요
	- 충전소 쏠림현상 완화, 충전 차량 적절히 분배, 충전 대기시간 감소
	- 충전소 충전 대기 현황을 제시하여 충전 대기기간 확인 가능
	- 자율주행으로 이동하여 충전하기 떄문에 실제 차주의 충전 충전시간, 충전대기시간 대폭 감소
	- 차주의 충전소 이동이 불필요하기 때문에 충전소때문에 일어나는 불편함 감소

▶ 원리
	- 지도에서 충전소 위치와 이용가능 충전기 실시간 확인 (지도 API)
	- 등록된 차량이 충전소로 출발
	- 잘 이동하고 있는지 실시간 랜더링 확인 (배달의 민족, 카카오 or Tmap 네비게이션 API이용)
	- 충전소에 도착할 차량들 충전소에서 리스트 확인
	- 리스트에 존재하는 차량들이 충전소로 도착하면 충전 진행
	- 충전이 완료되면 주유소에서 출발 인가를 내려 목적지로 출발


▶ 홈페이지

	- 회원가입 / 로그인 / 지도 / 문의 게시판
	- 오늘 이용가능한 충전소 충전기 지도에서 확인
	- 회원가입에서 일반 사용자와 충전소 사업자를 나눔


▶ 일반 사용자의 서비스 이용 방법

	- 문의 게시판 / 지도 / 내정보 - (수정하기 / 로그아웃)

	- 자동차면허증 등록 밎, 이용할 때마다 등록증, 본인 확인, OAuth 인증을 통한 회원가입
	- 출발지, 목적 충전소, 도착지, 출발 시간 설정 후 출발 시간이 되었을 때 알람을 주어 출발 허용을 통해 이용
	- 충전기 문제 시 신고, 이용가능한 충전소 충전기 갯수 1개 감소 진행 
 	
▶ 충전소 사업자의 서비스 이용 방법

	- 계약, 아이디 제공
	- 매일 이용 가능한 충전기 업데이트
	- 신고 시 이용 가능한지 확인
	- 이용 불가할 경우 충전기 비활성화 밎 이용 불가능한 충전기 정보에 올리기 -> 업데이트
	- 충전 요청 리스트에 있는 차량을 충전기에서 충전
	- 충전 완료 시 충발


▶ 추가 아이디어
	- 갑지기 충전소로 차량이 한번에 몰리는 것을 막기 위해 그 충전소로 몇명의 이용자들이 목적지로 지정했는지 실시간 랜더링
	- 차량이 길가에 서면 안되므로 충전소마다 구체적으로 잠깐 추자가 가능한 공간 배치
	- 주차 장소에 있는 차량에서 충전기까지 이동을 위해 운전소 직원에게 잠깐의 운전권한을 허용
